# ProteinSnap — Legal & Support

This repository hosts the public legal and support pages for the ProteinSnap iOS app — the Privacy Policy, Terms of Service, and Support page — served as a static site via GitHub Pages at https://easyprotein.github.io/proteinsnap-legal/. The pages are plain, self-contained HTML with inline styles, no external resources, no JavaScript, and no build step: edit the `.html` files directly and commit to publish.
